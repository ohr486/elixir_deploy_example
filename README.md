ElixirDeployExample
===================

# setup server(install elixir and ruby)
```
git clone git@github.com:ohr486/elixir_deploy_example.git
cd /elixir_deploy_example
cd setup
ansible-playbook setup.xml -i hosts
[wip] add ruby installation script to setup.xml
```

# setup local
```
bundle install
mina setup
```

# deploy
```
mina deploy
```
